API Basejump: Image Search Abstraction Layer

User stories:
    1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?offset=2 parameter to the URL.
    3) I can get a list of the most recently submitted search strings.

Example usage:

https://mat885-isal-freecodecamp.glitch.me/api/imageSearch/robots?offset=1

Example output:

       
        {
         "url": "https://ichef.bbci.co.uk/news/660/cpsprodpb/1999/production/_92935560_robot976.jpg"
         "snippet": "Could robots be marking your homework? - BBC News"
         "thumbnail": "https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSqmiiixqjQ4lVNfTgXZB6EivtowcgJf548xr4aKKUGq0JtKSYweHz-g-nN"
         "context": "http://www.bbc.com/news/business-38289079"
        }
